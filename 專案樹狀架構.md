## 模組結構

### 1) `brainwave-backend`（API 與啟動層）

```text
brainwave-backend/
|-- pom.xml                                   # backend 模組依賴（含 Flyway）
`-- src/main/
    |-- java/com/brainwave/backend/
    |   |-- BrainwaveBackendApplication.java  # Spring Boot 啟動入口
    |   |-- auth/
    |   |   |-- AuthController.java           # 後台登入 API：/api/auth/login
    |   |   |-- AuthFrontController.java      # 前台登入 API：/api/front/auth/login
    |   |   `-- dto/
    |   |       |-- AuthLoginRequest.java     # 登入請求 DTO
    |   |       `-- AuthLoginResponse.java    # 登入回應 DTO（token + user）
    |   |-- config/
    |   |   |-- DefaultResultMapper.java      # 預設 Result 組裝（可替換為企業格式）
    |   |   `-- WebMvcConfig.java             # 註冊 AuthGuard 等 Interceptor
    |   |-- filter/
    |   |   `-- CorrelationIdFilter.java      # 注入 X-Request-ID、寫入 MDC 供日誌追蹤
    |   |-- health/
    |   |   `-- HealthController.java         # 健康檢查 API：/api/health
    |   |-- user/
    |   |   `-- UserController.java           # 使用者 CRUD/查詢 API：/api/users
    |   `-- member/
    |       `-- MemberController.java         # 會員 CRUD/查詢 API：/api/members
    `-- resources/
        |-- application.properties             # 共用預設設定（JPA/Flyway/CORS/Auth）
        |-- application-local.properties       # 本地環境覆寫設定（local DB）
        |-- application-prod.properties        # 正式環境覆寫設定（prod）
        |-- test_data_users.sql                # 使用者測試資料 SQL
        |-- META-INF/additional-spring-configuration-metadata.json # 自訂 app.* 設定鍵 metadata
        `-- db/migration/
            `-- V1__init_users_members.sql     # Flyway 初始 migration（建表）
```

    |       |-- UserUpdateRequest.java        # 更新使用者請求（不含密碼）

        |-- repository/MemberRepository.java  # 會員資料存取介面

        |-- vo/MemberVo.java                  # 會員回傳視圖模型

    |   `-- request/
    |       |-- SystemConfigRequest.java      # 新增/上傳設定請求
    |       |-- SystemConfigUpdateRequest.java# 更新設定內容請求
    |       `-- SystemConfigSearchRequest.java# 系統設定搜尋條件請求

## 設定與部署現況

- Profiles
  - `local`：使用本地資料庫（`application-local.properties`）
  - `prod`：使用環境變數注入（`application-prod.properties`）
- DB migration
  - Flyway 啟用，migration 路徑：`brainwave-backend/src/main/resources/db/migration`
- 容器部署
  - 使用根目錄 `Dockerfile` + `.dockerignore`
- 測試資料
  - `scripts/seed-users-members.sql`
  - `brainwave-backend/src/main/resources/test_data_users.sql`

## API 與認證現況

- 後台登入：`POST /api/auth/login`
- 前台登入：`POST /api/front/auth/login`
- 健康檢查：`GET /api/health`
- 管理 API：`/api/users`、`/api/members`
- 目前為輕量 token/mock auth 設計，尚未接 Spring Security/JWT 驗簽流程
