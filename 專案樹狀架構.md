## API 與認證現況

- 認證是「輕量 mock token + 帳密驗證」
  - 後台：`POST /api/auth/login`
  - 前台：`POST /api/front/auth/login`
- 目前未接 Spring Security / JWT 驗簽層（後續可擴充）

## 設定與部署現況

- `application.properties`：共用預設（DB、JPA、mock auth）
- `application-local.properties`：本機資料庫
- `application-zeabur.properties`：Zeabur 覆寫入口（可載入本機私有檔）
- `Dockerfile`：Zeabur 多模組建置部署
- `scripts/*.sql`：初始化/測試資料腳本
