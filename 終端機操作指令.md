# 終端機操作指令

## 編譯與執行

```bash
# 編譯整個專案
mvn clean install

# 執行核心模組
cd brainwave-core
mvn spring-boot:run

# 執行前端 API 服務
cd brainwave-backend
mvn spring-boot:run
```

## 格式化（Spotless）

```bash
# 檢查格式（不修改檔案）
mvn spotless:check -DskipTests

# 自動修正格式
mvn spotless:apply -DskipTests

# 在 CI/CD 中使用 profile 進行格式檢查（會中斷建置）
mvn verify -Pspotless
```
