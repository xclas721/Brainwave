# Flyway 資料庫遷移說明

依版本順序執行：V1 → V2 → V3。每個檔案對應一個功能／階段，之後新增表或改 schema 請加 V4、V5…。

| 版本 | 檔案 | 說明 |
|------|------|------|
| V1 | `V1__init_users_members.sql` | 基礎帳號表：users、members |
| V2 | `V2__audit_log.sql` | 稽核日誌表（見 稽核日誌說明.md） |
| V3 | `V3__system_config.sql` | 系統設定 Key-Value 表（見 系統設定API說明.md） |

- 路徑：`brainwave-backend/src/main/resources/db/migration`
- 勿修改已部署過的 migration 內容；新變更請用新版本號。
