# 終端機操作指令

## 編譯與執行

```bash
# 編譯整個專案（含測試）
mvn clean install

# 執行後端 API（預設 local profile）
cd brainwave-backend
mvn spring-boot:run

# 以 JWT 驗證啟動（需設定 app.auth.jwt.secret）
mvn spring-boot:run -Dspring-boot.run.profiles=local,jwt
```

注意：可執行的為 **brainwave-backend**（內嵌 Spring Boot），brainwave-core 為程式庫模組，不單獨執行。

## 格式化（Spotless）

```bash
# 檢查格式（不修改檔案）
mvn spotless:check -DskipTests

# 自動修正格式
mvn spotless:apply -DskipTests

# 在 CI/CD 中使用 profile 進行格式檢查（會中斷建置）
mvn verify -Pspotless
```

## 測試

```bash
# 執行所有模組測試（建議從專案根目錄執行）
mvn test

# 僅執行 backend 模組測試
cd brainwave-backend
mvn test
```
